## Overview
An interactive web portal for automating various manual processes done by librarian.
## Core Features

- **Searching** of books
- **Issuing** and **returning** books
- Librarian/ admin functions:
  - **manage fines**(max KES 500 per member)
  - **read** information about any member
  - **track** the books issued by a particular member
  - **add/remove/edit** any member.
  - **add/delete/edit** books.
  - **update the availability status** of the books

## Additional Features

- **admin dashboard** with charts & reports.
- **add/delete/edit** book racks & shelves.
